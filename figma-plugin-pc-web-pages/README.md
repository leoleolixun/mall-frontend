## 使用方式

### 网页版 Figma

1. 打开 Figma 网页版，并打开文件 `商城项目全端原型图`。
2. 打开或选中 `02 前台三端原型` 页面。
3. 点击左上角 Figma 主菜单。
4. 进入 `Plugins > Development > Import plugin from manifest...`。
5. 选择本目录下的 `manifest.json`：

   `/Users/leo/GoWorkSpace/mall/mall-frontend/figma-plugin-pc-web-pages/manifest.json`

6. 导入后运行 `Plugins > Development > Mall PC Web Pages Builder`。

### 桌面版 Figma 兜底

如果网页版看不到 `Development` 或 `Import plugin from manifest...`，使用 Figma 桌面版执行同样步骤：

1. 打开 Figma 桌面版，并打开文件 `商城项目全端原型图`。
2. 进入 `Plugins > Development > Import plugin from manifest...`。
3. 选择：

   `/Users/leo/GoWorkSpace/mall/mall-frontend/figma-plugin-pc-web-pages/manifest.json`

4. 运行 `Plugins > Development > Mall PC Web Pages Builder`。

## 运行结果

- 插件会自动查找 `02 前台三端原型` 页面。
- 运行完成后，画布会出现 `PC Web 完整页面补充` 说明条、9 个 `PC Web group / ...` 分组标题条和 58 个 `PC Web / ...` 页面框。
- PC Web 页面会按业务模块分区，每个分组单独占一排，组内页面横向连续排布，便于一次看完同一业务链路：
  - `01 频道入口`：首页、分类、店铺、品牌、秒杀、新品等入口页。
  - `02 商品浏览`：列表、详情、评价、对比、预售、组合套餐。
  - `03 购物交易`：购物车、登录、订单确认、自提、支付结果。
  - `04 订单中心`：订单列表、空状态、取消、退款、详情。
  - `05 会员资产`：个人中心、资料、地址、优惠券、收藏、足迹、评价、积分。
  - `06 账号与消息`：安全、实名、支付密码、隐私、手机绑定、重置密码、消息、通知偏好。
  - `07 履约与售后`：物流、售后列表、售后申请、售后详情、退货物流。
  - `08 客服与发票`：工单、工单详情、发票管理、发票详情、注销账号。
  - `09 帮助与异常`：帮助中心、到货提醒、搜索空结果、404。
- 如果页面很多，Figma 可能需要几秒钟完成绘制，等待右下角插件提示结束即可。

## 导入前检查

可以先在项目根目录运行：

```bash
node mall-frontend/figma-plugin-pc-web-pages/check-plugin.js
```

看到全部 `PASS` 后，再导入 `manifest.json`。

## 常见问题

- 看不到 `Development` 菜单：优先改用 Figma 桌面版；开发插件入口在网页端可能不可用或不可见。
- 运行后没有生成在当前页：插件会优先写入名为 `02 前台三端原型` 的页面；如果找不到该页面，才会写入当前页。
- 想重新生成：直接再次运行插件即可，它会先清理自己生成过的 58 个页面、9 个分组标题条和说明条。

## 行为说明

- 插件会优先查找名为 `02 前台三端原型` 的页面，找不到时才使用当前页面。
- 插件可以重复运行。每次运行都会先删除同名的 58 个补充页面、9 个分组标题条和说明条，再重新生成，避免重复堆叠。
- 插件不调用 Figma MCP，不消耗 MCP 免费版额度。
